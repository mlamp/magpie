# magpie-bt-wire

Peer wire protocol codec for the [magpie](https://github.com/mlamp/magpie) BitTorrent library. Implements the BEP 3 wire protocol, BEP 6 Fast extension, BEP 9/10 extension protocol, and BEP 52 v2 wire additions.

**Status**: pre-M0. Empty placeholder; real implementation lands in M1.
